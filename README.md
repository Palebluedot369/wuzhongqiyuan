# 《物种起源》复刻

## 项目简介
本项目是独立复刻挂机类游戏《物种起源》核心数值系统的Unity工程。目前实现了粒子物种进化链、尘埃物种进化链、雷电双路线升级等核心玩法，所有数值采用数据驱动架构（JSON配置），具备完整的资源生产、解锁、合成、升级逻辑。

## 技术栈
- **引擎**：Unity 2022.3.62f1c1
- **语言**：C#
- **数据存储**：JSON
- **版本控制**：Git

## 核心功能
- [x] 21种粒子物种进化链（解锁、生产效率升级、批量合成、合成等级升级）
- [x] 11种尘埃物种进化链（同上）
- [x] 雷电双路线升级（混沌：加固定值；物质：加百分比）
- [x] 数据驱动架构（所有数值从JSON读取，无需修改代码）
- [x] 单例资源管理器 + 事件驱动UI刷新
- [x] 动态滑动列表（ScrollRect + LayoutGroup）
- [x] 合成弹窗（滑动条批量合成、科学计数法显示）

## 如何运行
1. 使用 Unity 2022.3 或更高版本打开本项目。
2. 打开 `Assets/Scenes/startGame.unity` 场景。
3. 点击运行按钮，即可开始游戏。
